This is the game called tic tac toe. In this game you can draw the shape of "O" or "X" by chicking into that red square. You can not change the sign in the square if you already filled it by clicking before. If the 3 boxes have same sign and if those 3 boxes are in the same line than that player will win. If this thing does not happens then no one will win and you should click "Start Again" button.
